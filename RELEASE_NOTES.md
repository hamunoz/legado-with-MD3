## ML Kit Translation Integration v1.0.0

### Release Notes
1. **ML Kit Translation base implementation with offline/online support**: Enabled translation functionality while allowing users to choose between online and offline modes.
2. **Database persistence**: Implemented automatic fallback from offline to online translation ensuring a seamless user experience.
3. **Automatic source language detection**: Leveraged ML Kit Language Identification to automatically detect the source language, improving the translation accuracy.
